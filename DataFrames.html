DataFrames (marcos de datos)

Creación de DataFrames

Una forma de crear DataFrames en R es mediante la función data.frame, como se muestra a continuación

Código 
set.seed(1234)
# 10 datos simulados de una variable aleatoria binmial con n=20 y p=0.4
x<-rbinom(10,20,0.4)  
# vector de caracteres aleatorios  
y<-sample(c("F","M"),10,replace=TRUE)
# vector logico 
z<-runif(10)>0.5

A continuación creamos un DataFrame con estos tres vectores

Código 
datos=data.frame(x,y,z)
datos
    x y     z
1   5 M FALSE
2   9 M FALSE
3   9 M FALSE
4   9 F FALSE
5  10 M FALSE
6   9 M  TRUE
7   3 M  TRUE
8   6 F  TRUE
9   9 M  TRUE
10  8 M FALSE

Para saber si efectivamente un objeto en R es un DataFrame podemos ejecutar los siguientes comandos

Código 
class(datos) 
[1] "data.frame"
Código 
is.data.frame(datos) 
[1] TRUE
Código 
mode(datos) 
[1] "list"

datos=data.frame(Cx=x,Cy=y,Cz=z)

Nombres de filas, usos

Los nombres de filas se pueden usar para recuperar filas del data (indexamos el data con un vector de caracteres que contiene los nombres de las filas que quiere recuperar)

Código 
datos=data.frame(Cx=x,Cy=y,Cz=z)  
datos[c('Jan','Mar','May'),] 
    Cx Cy    Cz
Jan  5  M FALSE
Mar  9  M FALSE
May 10  M FALSE

Recuperacón y cambio de nombres de filas y columnas

Para recuperar los nombres de filas de un data usamos la función row.names o rownames y para las columnas usamos colnames, de la siguiente forma

Código 
# filas 
row.names(datos) 
 [1] "Jan" "Feb" "Mar" "Apr" "May" "Jun" "Jul" "Aug" "Sep" "Oct"
Código 
rownames(datos)
 [1] "Jan" "Feb" "Mar" "Apr" "May" "Jun" "Jul" "Aug" "Sep" "Oct"
Código 
# columnas 
colnames(datos) 
[1] "Cx" "Cy" "Cz"

Recuperar nombres de columnas

Código 
# columnas 
colnames(datos) 
[1] "Cx" "Cy" "Cz"

Para cambiar los nombres de columnas procedemos de la siguiente forma

Código 
colnames(datos)=c('Colx','Coly','Colz')

datos$Cx2=datos$Colx^2

Indexando Data Frames

Como con los vectores y matrices, podemos indexar un data frame usando como indices

  • Enteros positivos
  • Enteros negativos
  • Valores lógicos
  • Caracteres

Esto es válido tanto para filas como para columnas

Veamos algunos ejemplos

Código 
datos[c(1,5,7,9,10),c(2,3)]
    Col2  Colz
Jan    M FALSE
May    M FALSE
Jul    M  TRUE
Sep    M  TRUE
Oct    M FALSE

Indexado con numeros negativos

Código 
datos[-c(1,5,7,9,10),-c(2,3)]
    Colx Sex
Feb    9   1
Mar    9   1
Apr    9   0
Jun    9   1
Aug    6   0

Ejemplo de indexado con caracteres

Código 
datos[c('Feb','Apr','Jun','Aug'),-1]  
    Col2  Colz Sex
Feb    M FALSE   1
Apr    F FALSE   0
Jun    M  TRUE   1
Aug    F  TRUE   0

Indexación lógica

En este tipo de indexación se extraen las filas o columnas que coincidan con un valor TRUE , veamos ejemplos

Código 
datos[c(TRUE,FALSE,TRUE,FALSE,TRUE,FALSE,TRUE,FALSE,TRUE,FALSE), ]  
    Colx Col2  Colz Sex
Jan    5    M FALSE   1
Mar    9    M FALSE   1
May   10    M FALSE   1
Jul    3    M  TRUE   1
Sep    9    M  TRUE   1
Código 
datos[!c(TRUE,FALSE,TRUE,FALSE,TRUE,FALSE,TRUE,FALSE,TRUE,FALSE), ]  
    Colx Col2  Colz Sex
Feb    9    M FALSE   1
Apr    9    F FALSE   0
Jun    9    M  TRUE   1
Aug    6    F  TRUE   0
Oct    8    M FALSE   1

Ejercicio

Ejercicio: Calcular la media de Colx pero para aquellas filas conde Col2 es igual a “M” y separadamente para donde Col2 es igual a “F”

Código 
datosM=datos[datos$Col2=="M",] 
mean(datosM$Colx) 
[1] 7.75

Para los femeninos, es decir, donde Col2 es igual a “F”

Código 
datosF=datos[datos$Col2=="F",]  
mean(datosF$Colx) 
[1] 7.5

Otra forma, usando la negación

Código 
datosF=datos[!datos$Col2=="M",]  
mean(datosF$Colx) 
[1] 7.5

Ejercicio

Calcular la varianza de Colx para aquellas filas donde Colz es TRUE

Código 
var(datos[datos$Colz,][["Colx"]])

Ejercicio

Calcular la varianza de Colx para aquellas filas donde Colz es FALSE

Ejercicio

Calcular la media de Colx para equellas filas donde Col2 es ‘M’ y Colz es FALSE

Código 
mean(datos[datos$Col2=='M' & !datos$Colz, ][['Colx']])
[1] 8.2
